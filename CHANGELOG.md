# Changelog

## 1.1.0

- Add **Open Locally With...** on Windows to show the OS app picker for mapped local paths.
- Fall back to the selected local path when no mapping matches a local `file:` resource.
- Add more verbose diagnostics for mapping and open failures.

## 1.0.0

- Add Explorer commands for opening, revealing, and copying mapped local paths.
- Add editable Settings UI mapping configuration.
- Run as a local UI extension for Remote and Dev Container workspaces.
- Add GitHub Actions CI packaging workflow.

## 0.0.1

- Initial release.
- Add Open Locally, Reveal Locally, and Copy Local Path Explorer commands.
- Add configurable remote-to-local path mappings.
